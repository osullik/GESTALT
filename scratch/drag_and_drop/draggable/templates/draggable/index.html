GESTALT - Geospatially Enhanced Search with Terrain Augmented Location Targeting
Select a region to search, then add objects to the query canvas using the dropdown menu to the right. Toggle between "Object-centric" or "Location-centric" search mode, and specify whether your query is aligned with the cardinal directions. Arrange the objects on the canvas for GESTALT to search. A successful query will return a list of locations in the region matching the spatial configuration of objects on the query canvas.
